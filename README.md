# zsh-up

ZSH integration for the [Ultimate Plumber][].

[Ultimate Plumber]: https://github.com/akavel/up

## Installation

### zplug

```
zplug "utdemir/zsh-up"
```

### Antigen

```
antigen bundle "utdemir/zsh-up"
```

### Manual

Download zsh-up.plugin.zsh and source it from your ~/.zshrc file
